# WS Store APP

Bot resmi WS Store Official untuk membantu transaksi Discord store secara otomatis.

Fitur utama:
- Verification system untuk member baru
- Ticket order, support, dan rekber / middleman
- Payment QRIS panel
- Invoice otomatis ke DM pembeli
- Transaction / vouch otomatis
- Customer tier otomatis berdasarkan total order
- Anti-spam moderation dengan warning dan timeout

WS Store APP dibuat untuk menjaga proses order tetap cepat, rapi, aman, dan mudah dipantau oleh staff.
